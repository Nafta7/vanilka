# Vanilka

Vanilka is an wordpress starter theme.

## Features

- Sass
- ES2015
- Live-reload
- Modular *functions.php*

## Install

```bash
cd path/to/wp-site/wp-content/themes
git clone https://github.com/Nafta7/vanilka.git theme-name
cd theme-name
npm install
npm build
```

## Develop

```bash
npm start
```

Now it's ready to edit the new template.
