Vanilka

a minimal Wordpress starter theme.

Download

Minimal

Does not include any framework or library like Bootstrap and jQuery. Only pure, lean markup.

Modular

Because the theme was build with modularity in mind, it's very easy to enable or disable theme features.

Modern

Write the latest version of JavaScript paired with Sass and support for live-reload on the browser.

Features

A clean head

Removes all the junk that Wordpress puts into the markup by default.

Helpful partials to get you started

A couple of partials (or template-parts like Wordpress calls it) are available to help you get started. The templates are neatly organized by post, page, and sections. Check the partials directory.

You also have a custom page template under the template directory.

ES2015 and Sass

Write the latest version of JavaScript (ES2015) paired with Sass.

Modern workflow

The asset files you are working on are automatically compiled with support for live-reload on the browser as you saves them.

Manage scripts and front-end dependencies with npm.

Customize

Easily customizable to fit your needs.

Selected Wordpress theme features

Enable only the most basic and essential Wordpress features needed to build a website today. The following features are enabled by default:

  • Menu: Register a menu for the main navigation of the website.
  • Widgets: Enables widgets with a sidebar by default.
  • Featured image: Enabled by default.
  • Assets: Enqueues a stylesheet file and a JavaScript file.
